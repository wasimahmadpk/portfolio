## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (if you want to clone the repository)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/wasimahmadpk/portfolio.git
cd portfolio
```

2. Open `index.html` in your web browser:
   - Simply double-click the file, or
   - Use a local server (recommended):
```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

3. Visit `http://localhost:8000` in your browser

## ğŸ¨ Customization

### Updating Content

- **Personal Information**: Edit the hero section and contact details in `index.html`
- **Experience**: Update the timeline items in the Experience section
- **Education**: Modify the education cards

- **Colors**: Modify CSS variables in `styles.css` (look for `:root` selector)
- **Fonts**: Change the Google Fonts import in `index.html`
- **Layout**: Adjust spacing, sizes, and grid layouts in `styles.css`
